Reciclaje de vidrio

Para poder llevar a reciclar principalmente las botellas de vidrio, estas ya sean de jugo, bebida, agua, cerveza, destilados, vino y/o espumantes.

Se debe realizar el siguiente procedimiento:

  • Enjuagar y/o lavarlas una vez vacias
  • Secarlas o dejas estilando
  • Apilarlas en una caja ya sea de plastico o carton para llevarla al punto limpio
  • En caso de que se hayan roto se recomienda llevarlas dentro de una caja de carton para evitar cortes
